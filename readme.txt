操作説明書

1,まず下のアカウント登録ボタンから、アカウント登録をお願いします。

2,登録について ユーザ名は20文字以下、パスワードは4文字以上が必須です。

3,登録が完了するとログインページに遷移します。登録したユーザIDとパスワードでログインしてください。

4,投稿には画像,撮影場所,使用機材,各種設定と撮影日時が必須です。メモは任意なのであってもなくても投稿できます。

5,投稿されると左側に時系列順に並んでいきます。削除したい場合は削除ボタンを押して、ホップアップを確認してください。

6,サインアウトしたい場合は左上にボタンがあります。
